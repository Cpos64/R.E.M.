# R.E.M. - Sleep and Dream Social App 🌙

A Flutter application focused on promoting healthy sleep and dream journaling, built with Firebase.

## 📖 Description

R.E.M. (Rest Experience Mindfulness) is a social media app designed for athletes and active individuals to track their sleep, journal their dreams, and share insights with their community. 

## 🚀 Features

- ✅ Firebase Authentication (Signup, Login)
- ✅ Dream Journaling
- ✅ Sleep Tracking
- ✅ Firebase Firestore Integration
- ✅ Multi-Platform Support (Android, iOS, Web)

## 📦 Dependencies

- flutter
- firebase_core
- firebase_auth
- cloud_firestore
